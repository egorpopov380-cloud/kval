# Корочки.есть - Система управления заявками на обучение

## Описание проекта

Веб-приложение для управления заявками на обучение. Пользователи могут создавать заявки на курсы, оставлять отзывы, а администраторы — управлять статусами заявок.

## Технологии

### Backend
- Node.js
- Express.js
- PostgreSQL
- bcrypt (хэширование паролей)

### Frontend
- HTML5
- CSS3
- Vanilla JavaScript
